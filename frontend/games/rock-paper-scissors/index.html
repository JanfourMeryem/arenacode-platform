Rock Paper Scissors Challenge

Rock Paper Scissors Duel

Read the number of rounds from stdin, print one move per round, then run your code. The platform generates random bot moves each run and replays the duel.

Rules

  • Output exactly one move code per line.
  • Allowed move mapping:
1 = ROCK 2 = PAPER 3 = SCISSORS
  • ROCK beats SCISSORS.
  • SCISSORS beats PAPER.
  • PAPER beats ROCK.
  • Equal moves produce a draw.

Input / Output

Injected input:

  • n = number of rounds
5

Expected output: exactly n lines, each line in {1,2,3}.

1
2
3
1
2

Example

# PSEUDO-CODE (not runnable)

repeat n times:
    choose your move with your own strategy
    output one value among {1, 2, 3}
