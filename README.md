# ArenaCode Platform

ArenaCode is a gamified educational platform designed to help users learn and practice programming through interactive challenges and real-time code execution.

## 🚀 Features

- 📘 Learn module for structured programming content  
- 💻 Interactive Playground with real-time code execution  
- 🎮 Gamified Challenges to apply algorithms  
- ⚡ WebSocket-based real-time communication  
- 🔒 Secure code execution using Docker containers  

---

## 🏗️ Architecture

The platform follows a client–server architecture:

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Node.js with Express  
- **Execution Engine**: Docker-based isolated environment  
- **Reverse Proxy**: Nginx  
- **Communication**: REST API & WebSocket  

---

## 🛠️ Technologies Used

- HTML / CSS / JavaScript  
- Node.js / Express  
- Docker  
- Nginx  
- Monaco Editor  
- xterm.js
